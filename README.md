# Cursur 브레이즈 팝업 빌더 테스트

## ⚠️ 중요: 프로젝트 경로에 `#` 사용 금지

**경로에 `#`가 있으면 Vite가 파일을 찾지 못합니다.**  
예: `.../ #진행 중인 업무/브레이즈_Cursor 테스트` → `#` 제거 후  
`.../진행 중인 업무/브레이즈_Cursor 테스트` 처럼 폴더 이름을 바꾼 뒤 실행하세요.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속.

## 구성

- **왼쪽 (Control Panel)**  
  배경 이미지(업로드/URL), 버튼 개수(1개/2개), 버튼 스타일(색상·텍스트), 오버레이 투명도, 모서리 둥글기(R) 설정

- **오른쪽 (Preview)**  
  아이폰 비율(390×844) 미리보기. 1:1 팝업, `object-cover` 배경, 하단 고정 "Don't show again" / "Close X"

- **하단**  
  설정값 실시간 JSON 출력 (이미지 주소 포함, Base64는 `[Base64 Image]`로 표시)

## 이미지

- **파일 업로드**: Base64로 변환 후 즉시 미리보기 배경에 적용
- **URL 입력**: 주소 입력 후 적용 버튼으로 반영

## 빌드

```bash
npm run build
```

빌드 결과는 `dist/` 폴더에 생성됩니다.
