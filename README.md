# Cursor 브레이즈 팝업 빌더 테스트

MCP 테스트를 위한 브랜치 입니다.

---

## 🚀 1. 시작하기

1. **프로젝트 열기**: Cursor 실행 후 폴더를 선택해 엽니다.

2. **터미널 열기**: 키보드에서 `Ctrl` + ` ` ` (물결 키)를 누릅니다.

3. **초기 설정**: 아래 명령어를 복사해서 터미널에 넣고 엔터를 누르세요.

```
npm install 
```

---

## 💻 2. 실행 및 확인

1. **개발 서버 실행**: 아래 명령어를 터미널에 입력하세요.

```
npm run dev
```


2. **브라우저 접속**: 터미널에 뜬 `http://localhost:3000` 주소를 클릭하세요.

---

## 📤 3. 작업 전달하기 (Git)

수정이 끝나면 아래 3개를 순서대로 터미널에 입력하세요.

**① 파일 담기**
```
git add .
```

**② 메모 남기기**
```
git commit -m "디자인 수정 (예시입니다)"
```

**③ 서버 전송**
```
git push origin test-MCP
```

---

## 📌 주의사항

- 화면 왼쪽 아래가 **test-MCP** 인지 꼭 확인하세요!

- 작업 시작 전에는 항상 아래 명령어를 먼저 입력하세요.

```
git pull origin test-MCP
```
